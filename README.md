# Проект "Научиться учиться"

## Описание проекта и его функциональности
В проекте рассказывается, какие современные и эффективные подходы к обучению можно использовать в своей жизни.

### Разделы сайта содержат:
- Описание 5 техник для эффективного обучения
- 2 видео - о прокрастинации и подходе к собственному обучению
- Историю успешного применения метода обучения Ричарда Фейнмана
- Немного цифр и фактов об учебе и мозге
- Ссылку на книгу, которую стоит прочитать каждому, кто занимается образованием
- 10 принципов обучения от Джоша Кауфмана
- Ссылки на несколько полезных ресурсов

Сайт служит примером верстки с помощью flex, использования медиаконтента в iframe, различных анимаций в текстах, ссылках и изображениях.

Структура проекта построена на методологии БЭМ.
